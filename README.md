# MICRO-MANIA
Collection of 2 player mini-games played over Wi-Fi using ARM Cortex-M4 processor and custom RTOS.

## Game Collection
Slither.io

Flappy Bird

Tiles (_Single-player memory tile game_)

Goomball (_Mario-themed dodgeball game_)
